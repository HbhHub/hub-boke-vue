###### hsdb工具

```
推荐使用 JDK自带的hsdb工具查看Java虚拟机内存信息。工具位于JDK安装目录下lib文件夹中的sa-jdi.jar中。
⚫启动命令：java -cp sa-jdi.jar sun.jvm.hotspot.HSDB
```
